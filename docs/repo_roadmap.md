# Repository Roadmap

This document describes what the repository is prepared for before the data science implementation begins.

## Ready now

- Python package layout
- configuration files
- locked specification documents
- dataset placement guidance
- source package scaffolding
- basic test scaffolding

## Intentionally deferred

- FreiHAND parsing implementation
- projection and geometry code
- actual HandsomeNet model layers
- training loop
- webcam inference path
- gesture recognition

These remain deferred because the first required implementation checkpoint is dataset integration correctness.
